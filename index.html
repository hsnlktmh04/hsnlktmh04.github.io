Selamat Datang di Husnul Library!

Halo! Saya adalah mahasiswa Program Studi Ilmu Perpustakaan dan Sains Informasi di Universitas Sumatera Utara. Website ini dibuat sebagai bagian dari pemenuhan tugas Mata Kuliah Pengembangan Web. Melalui proyek ini, saya berusaha untuk mengasah keterampilan dalam pengembangan website, terutama dalam hal desain, fungsionalitas, dan pengalaman pengguna agar lebih interaktif dan informatif. Saya berharap website ini tidak hanya menjadi tugas akademik semata, tetapi juga dapat menjadi media pembelajaran yang bermanfaat. Ke depannya, saya ingin terus mengembangkan dan menyempurnakan website ini agar dapat memberikan nilai tambah serta pengalaman yang lebih baik bagi para penggunanya.
